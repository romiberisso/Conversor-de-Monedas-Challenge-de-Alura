# Conversor-de-Monedas-Challenge-de-Alura
Challenge de Alura LATAM, creando un convertor de monedas con el cual se usa en tiempo real ExchangeRate-API.

Como adquirir Tu API Key

Visitando la pagina: "https://www.exchangerate-api.com", e ingrensando tu email, se puede adquirir una API Key gratis. 
Esta sera necesitada para el funcionamiento del programa.
La API Key debera ser copiada y pegada en la sigueinte linea, dentro del programa:

******  String apiKey = "TU-API-KEY" ******
